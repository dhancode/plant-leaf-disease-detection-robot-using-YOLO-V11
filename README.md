# AI-Powered Smart Agriculture Robot using YOLO, ESP32 and Firebase

## Overview
This project is an AI-powered smart agriculture robot capable of real-time plant disease detection using YOLOv8 integrated with ESP32 and Firebase.

The robot supports both manual and automatic navigation modes while performing real-time disease monitoring in agricultural environments.

## Features
- Real-time plant disease detection
- YOLOv8-based object detection
- ESP32 robot control
- Firebase cloud integration
- Manual and automatic navigation modes
- RTSP live stream support
- Real-time disease monitoring and logging

## Technologies Used
- Python
- YOLOv8
- OpenCV
- PyTorch
- Firebase Realtime Database
- ESP32
- Arduino IDE

## Project Structure
- dataset/
- ai_model/
- python_scripts/
- esp32_code/
- notebook/
- screenshots/

## System Architecture
1. ESP32 controls robot movement.
2. Firebase stores robot commands and detection data.
3. Python scripts process RTSP video stream.
4. YOLOv8 performs disease detection.
5. Detection results are uploaded to Firebase.
